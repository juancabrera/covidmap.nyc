Privacy Policy

NYC COVID map is a voluntary and anonymous location based mobile app designed to provide you with the latest COVID data around you in order to protect yourself and others from COVID-19.

The use of this app is completely voluntary, and it can be downloaded for free on iOS and Android smartphones.

Privacy is protected by not saving or sending your location data. Location data is only used to detect if you move from one zip code to another and notify you with the latest COVID data of your current zone. You can still use the app without granting location access permissions.

Independent project

NYC COVID map was designed and developed 100% by two individuals as a personal side project, there are no affiliations to any private or public organizations.

Data

The app uses only official data released by “The City of New York”:

  • Zip Code Tabulation Areas, from NYC OpenData
  • COVID-19 daily data, NYC Health Github

If location access permission is granted, the app will check your location approximately every 15 minutes to check if you moved into another zip code area, if so, the app will send you a “location changed” notification with the latest COVID data of the current zip code area.

If location access permission is declined, you can still use the app and see latest COVID data in other zip codes areas by taping them, you won’t be able to get any notifications.

Location data is not saved or sent anywhere, it’s only used for notifications purposes.
